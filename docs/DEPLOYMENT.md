# 🚀 Deployment & Hosting Guide

## Purpose
Provides standard steps for deploying the system to production.

---

## Server Requirements
- Linux (Ubuntu LTS)
- PHP 8.3
- MySQL / MariaDB
- Nginx or Apache
- Node.js

---

## Deployment Flow
1. Clone repository
2. Install dependencies
3. Configure environment
4. Run migrations
5. Build assets
